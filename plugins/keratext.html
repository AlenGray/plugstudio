Features

▣ Background Box

Core
  • Auto-fit, Square, Polygon (3–32 sides)
  • Per-side expansion control
  • Uniform expansion mode
  • Per-corner roundness control
  • Gradient fill (Linear / Radial)
  • Adjustable opacity
  • Ascender / Descender override

◫ Border / Stroke

Core
  • Customizable width, color, opacity
  • Cap style (Butt, Round, Square)
  • Trim Path (start, length, offset)
  • Dash pattern (length, gap, offset)
  • Per-side enable toggle
  • Gradient fill (Linear / Radial)
  • "Border over text" mode

◐ Gradient System

Visual
  • Two-color gradient fill
  • Linear and Radial modes
  • Adjustable angle and smoothness
  • Works on BG, Border, Highlight, Underline
  • Compatible with glow system
  • Start / End opacity control

✦ Outer Glow

Neon
  • Multi-pass neon glow rendering
  • Independent per component (BG, Border, HL, UL)
  • Radius, Spread, Intensity, Opacity
  • Gradient-safe color preservation
  • Additive composite for authentic neon
  • Matte-aware glow source

█ Highlight

100 Slots
  • Per-character selection (float — smooth edges)
  • Independent color, opacity, expansion per slot
  • Roundness control
  • Offset X/Y per slot
  • Gradient fill per slot (Linear / Radial)
  • Animation: Wipe L→R, Wipe R→L, Scale, Fade
  • Stagger & easing support

▁ Underline

100 Slots
  • Styles: Solid, Dashed, Dotted, Wave
  • Per-character selection (float)
  • Thickness, offset, cap style
  • Dash pattern (length, gap) + Trim Path
  • Rounding control
  • Gradient fill per slot (Linear / Radial)
  • Stagger animation & easing

⊞ Transform

  • Position offset (X/Y)
  • Scale (uniform & individual X/Y)
  • Rotation
  • Skew with offset
  • 9-point anchor control

◧ Matting

  • Matte to Fill — text shape reveals background
  • Subtract Text — text cuts holes
  • Subtract Border — border cuts holes
  • Works on Background, Highlight, Underline
  • Glow respects matting (matte-aware source)

⚡ Performance

  • SmartFX rendering pipeline
  • Threaded rendering support (AE 2022+)
  • 8 / 16 bpc native processing
  • 32 bpc float-aware compatibility
  • Unicode text (UTF-16)

Installation

1

Download

Purchase KeraText and download the installer or .aex file.

2

Install

Run Keratext_Setup.exe — it auto-detects your AE version. Or copy .aex to Plug-ins\KeraText\

3

Create

Restart After Effects, create a Text Layer, apply Effect → KeraText → KeraText.
